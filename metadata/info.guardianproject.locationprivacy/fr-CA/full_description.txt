GéoConfidentialité n'est pas vraiment une appli, mais plutôt un jeu de « filtres d'intention » pour toutes les différentes façons de partager un emplacement. Lorsque vous partagez votre emplacement à partir d'une appli, GéoConfidentialité est offert comme option. L'appli reconnaît ensuite les méthodes risquées de partage de l'emplacement et les convertit en des méthodes plus sûres. Ceci signifie principalement que l'appli réécrit les URL pour HTTPS, et même pour utiliser des URL `geo:` qui peuvent fonctionner avec des configurations entièrement hors ligne. GéoConfidentialité fonctionne principalement en lisant les informations d'emplacement de l'URL même. Pour de nombreuses URL, GéoConfidentialité doit en fait charger la page Web afin d'obtenir l'emplacement.

GéoConfidentialité peut aussi servir à réacheminer tous vos liens d'emplacement vers votre appli de cartes préférée. Toutes les applis de cartes pour Android peuvent visualiser les URL `geo:`, et GéoConfidentialité convertit de nombreux types de liens en URL `geo:`, incluant les cartes Google Maps, OpenStreetMap, Amap, Baidu Map, QQ Map, Nokia HERE, Yandex Maps.
